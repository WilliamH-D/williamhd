Cryogen

Started: Apr 2020 | Finished: Apr 2020

Ludum Dare Link: Cryogen

Controls: Movement - WASD, Shoot - LMB, Reload - R, Switch Weapon - Q, Interact - E

Instructions: Survive the waves of mutant hordes and protect the cryogen chamber by exploring the derelict mansion and placing down turrets to defend yourself!

Cryogen is a short first person shooter/tower defence survival game I made alongside a couple of friends for the 46th Ludum Dare group game jam. The game was made within 72 hours using the Unity game engine and programmed in C#.

The theme for the Ludum Dare 46 game jam was "Keep it alive". With only this theme and a short set of rules, we had 72 hours to brainstorm and produce a user experience that revolved around the theme. Hence we created a shooter game where you not only had to keep yourself alive, but you had to prevent the cryogen chamber from being destroyed by the mutants.

We initially used a GitHub repository to collaborate on the project, however, after an incident that cost us a few hours to resolve, we switched to Unity's built-in collab system.

This project has taught me a lot about working towards a strict deadline and collaborating with other developers to produce a product meeting specific guidelines.
